# MonMon Dashboard

Energy markets dashboard for oil brokers.

## Features
- Live crude oil prices (WTI & Brent)
- Refined products (RBOB Gasoline, Heating Oil)
- Calendar spreads
- Crack spreads
- Mobile-optimized interface

## Deploy to Render

This app is configured to deploy on Render's free tier.

## Local Development

```bash
npm install
npm start
```

Access at http://localhost:3456
